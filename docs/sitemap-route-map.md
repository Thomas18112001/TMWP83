# Sitemap and route map

- Accueil: `/`
- Le club: `/le-club`
- Les equipes: `/equipes`
- Equipe feminine Elite: `/equipes/feminine-elite`
- Competitions: `/competitions`
- Matchs, resultats, classement: `/competitions/matchs`

member profile, documents or admin workflows.
