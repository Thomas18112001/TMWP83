# Public site

Boundary marker for the public `toulonwaterpolo.fr` Next.js site.

The deployable source currently remains at the repository root:
- `app/`
- `components/`
- `lib/`
- `public/`
- `styles/`
- root Next/Tailwind/TypeScript config

Those files were not moved because `deploy.py` and the current deployment expect the root layout.
